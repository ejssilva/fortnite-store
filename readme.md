Duas telas do terminal abertas uma para rodar o server e outra para rodar client

Backend: ...\fortnite-store\server 
Comando: npm run start (Inicia a API e conecta ao banco de dados (na porta 5001)).

Frontend: ...\fortnite-store\client
Comando: npm run dev (Inicia o site em React (na porta 5173) e abre no navegador).

Tecnologias Utilizadas
Backend
    Node.js - para executar JavaScript no servidor. 
    Express.js - definir rotas (endpoints) e gerenciar requisições HTTP. 

Banco de dados
    MongoDB - NoSQL (orientado a documentos);

Segurança Atenticação 
    JSON Web Tokens (JWT): Usado para criar tokens de acesso seguros para os usuários após o login.

Frontend
    React: para construir interfaces de usuário baseada em componentes
    CSS: estilizar os componentes individualmente.
